# Contributing

Thank you for contributing to OCPP CP Admin.

## Licensing and traceability

By submitting code to this repository, you agree that your contribution is licensed under AGPL-3.0-only.

Please keep your Git identity configured correctly so authorship remains traceable:

- `git config user.name "Your Name"`
- `git config user.email "you@example.com"`

## Commit guidelines

- Use clear commit messages describing the intent.
- Keep commits focused and reviewable.
- Reference issues when relevant.

## Developer Certificate of Origin (recommended)

When possible, sign commits with:

- `git commit -s`

This adds a Signed-off-by line to improve contribution traceability.
